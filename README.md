# Timber E-commerce Database (Oracle)

## Project Overview
**Timber** is a sample e-commerce platform database designed for Oracle Database. It manages products, suppliers, customers, and orders. This project demonstrates:

- **Oracle database design** with tables, relationships, and constraints
- **Demo data** to simulate a real e-commerce environment
- **SQL reporting scripts** to analyze business insights, such as popular products, supplier performance, and sales trends

---

## Features
- **Database Build Scripts**  
  Create the Timber schema and tables with all necessary relationships.

- **Demo Data**  
  Populate the database with realistic sample data to simulate an operational e-commerce platform.

- **Reports / Analytics**  
  Prewritten SQL queries to generate reports such as:
  - Most popular products
  - Supplier performance metrics
  - Customer purchasing trends
  - Inventory and sales summaries

- **Schema Diagram**
  Visual representation of the database structure to help understand relationships.
